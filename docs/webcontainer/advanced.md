## Advanced Topics

### WASI support (2024)

WASI expands WebContainers to run WebAssembly modules requiring a system interface. Validate module compatibility and performance characteristics for your use case. [Announcing WASI](https://blog.stackblitz.com/posts/announcing-wasi/)

### Native package manager support (2024)

Improved native support for npm/yarn/pnpm enhances install speed and reliability. Prefer pnpm for efficient node_modules layout when possible. [Native PM support](https://blog.stackblitz.com/posts/announcing-native-package-manager-support/)

### Starter templates update (2024)

Updated starters streamline onboarding and demos; use them as baselines for custom UIs. [Starters update](https://blog.stackblitz.com/posts/webcontainers-starters-update/)

### Runtime test cases for AI agents

WebContainers provide deterministic, browser-contained environments suitable for evaluating tool-using agents and running integration scenarios without external VMs. [Runtime Test Cases for AI Agents](https://webcontainers.io/guides/runtime-test-cases-for-ai-agents)

### Versioning and support

Track API changes and support windows to ensure your app remains compatible across releases. [API versioning and support](https://webcontainers.io/guides/api-versioning-and-support)
